# Customer Components

UI components focused on displaying and managing customer-specific data.

## Files
- **`CustomerTable.js`**: The main data grid showing all customers. Includes sorting, pagination, and search.
- **`CustomerRow.js`**: An expandable row that shows a quick summary of recent transactions.
- **`TransactionRow.js`**: A small sub-component for individual transaction display.
- **`CustomerDetailsDialog.js`**: A modal showing the full 3-month reward breakdown for a selected customer.
